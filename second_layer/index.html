second layer

for the second layer all you have to do is find an edge piece that does not have your bottom colour (in this case yellow) on it and line up whatever colour isn't facing up with its center, then do this algorithm. if it needs to go in the left spot, Ui Li U L U F Ui Fi, and on the right, U R Ui Ri Ui Fi U F. if there are no edges on the top layer replace one of wrong pieces with one of the yellow pieces (if none of them are wrong then you are done the second layer!)
