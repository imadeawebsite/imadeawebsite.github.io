the cross

this is the first step in solving the four edge pieces on the top layer. this is a fairly intuitive process so I'll just go over it quickly. find an edge piece of your cross colour (I use white) and try to line it up with the white center. then, turn the top layer until the other colour lines up with one of the four centers. for the next edges you'll need to move the edge(s) you have already solved offset so that when you line up the new edge the old ones become resolved (this is a reoccurring concept.)
