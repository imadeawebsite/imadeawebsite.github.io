orient edges

this step is to get all the yellow (or last layer) edges facing up. to do this, there are two algorithms to do, each corresponding to a different case. you can either have no edges oriented, two oriented in an L shape, two oriented in a line, or four edges oriented. if you have four, your done this step! if you have two in a line, hold the cube so the line is horizontal and do F R U Ri Ui Fi. if you have two in an L shape, do F U R Ui Ri FI. and finally, if no edges are oriented, do one algorithm and the other two edge case will appear.
